正常返回

单体返回

http 200

集合返回

http 200

异常返回

通用异常返回

http 400

数据验证错误异常返回

http 403

错误码表（待修改）

错误码 错误描述
200 成功或正常
429 访问过于频繁
4000 未知错误
4001 未登录或登录状态已超时
4002 认证失败
4003 数据验证失败
4004 登录失败
4005 令牌过期
4006 令牌无效
4007 令牌数据错误
4008 没有权限
4009 令牌已被黑名单
  • 前端开发人员注意：
    前端开发人员无需枚举此码表的描述，因为此描述信息会在接口中返回。但开发人员需处理特定的错误码，例如：4005错误为令牌过期，对于此错误的处理应为刷新令牌。以此类推。
