Homestead

  1. git clone https://github.com/laravel/homestead.git Homestead
  2. bash init.sh // linux
  3. init.bat // windows
  4. vagrant box list 先确定已下载了laravel/homestead的box
  5. vagrant up
  6. vagrant ssh
